HEADINGS

CSS selector:

h1, h2, h3

Sample code:

<h1>Heading</h1>
<h2>Subheading 1</h2>
<h3>Subheading 2</h3>

PARAGRAPHS

CSS selector:

p

Sample code:

<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit...</p>

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Ut interdum, enim vel vehicula varius, sem arcu bibendum sem, ac fringilla felis arcu at justo. Nam quis eros ultricies, blandit sapien a, egestas ex.

BULLETED LISTS

CSS selector:

ul

Sample code:

<ul>
<li>Project 1<li>
<li>Project 2<li>
<li>Project 3<li>
</ul>
  • Project 1
  • Project 2
  • Project 3

NUMBERED LISTS

CSS selector:

ol

Sample code:

<ol>
<li>Project 1<li>
<li>Project 2<li>
<li>Project 3<li>
</ol>
  1. Project 1
  2. Project 2
  3. Project 3

IN-TEXT LINK

CSS selector:

.hyperlink, .hyperlink:active, .hyperlink:hover, .hyperlink:focus

Sample code:

<a href="#" class="hyperlink">In-text link</a>

In-text link

BUTTONS

CSS selector:

.button, .button:hover, .button:focus

Sample code:

<button type="button" class="button">Project 1</button>
